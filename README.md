# The Co Programming language
Co is a systemic approach to the programming language.

## The Co Programming Language syntax
```
"a 5        variable/array"
"< < >     enumerate/array"
"< > >    structure/object"
"< < > >   function/object"

b     "sol"
c "s ağ/ol"
ç     "sağ"
a b c ç

< color < red blue green >
< animal                 >
  kind  "cat"
  age     20
  says "myau"
>

x 10
y 20
< add < x y > x + y >

print < add < x y > >

< amra < >           >
< amra < x y >       >
< amra < x y > x + y >
< amra < x y         >
  x + y
>

yaz < amra < x y > >
yaz < "güle güle"  >
```
